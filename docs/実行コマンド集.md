### extract-output.mjs（jsonファイルからテストケースの抽出）

```shell
.\scripts\run-extract-outputs.ps1
```

出力先を作り直したい時

```shell
.\scripts\run-extract-outputs.ps1 -CleanOutput
```

**機能概要**

- results から results\extracted を作る

### validate-output.mjs（出力構造チェック）

```shell
.\scripts\run-validate-output.ps1
```

``

結果は .\reports\validate-output-summary.csv と .\reports\validate-output-summary.json に出る。

**機能概要**

- output.txt の形式を全件チェックして集計する

### validate-spec.mjs（内容カバレッジチェック）

```shell
.\scripts\run-validate-spec.ps1
```

「共通」を除外したいとき

```shell
.\scripts\run-validate-spec.ps1 -SkipCommonFeature
```

**機能概要**

- feature ごとの必須観点が入っているか全件チェックして集計する
